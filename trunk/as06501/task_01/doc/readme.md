## Практическая работа: Моделирование объекта управления

Рассматривается объект управления - температурная система, описываемая дифференциальным уравнением:

$$\Large\frac{dy(\tau)}{d\tau}=\frac{u(\tau)}{C}+\frac{Y_0-y(\tau)}{RC} $$ (1)

где $\tau$ – время; $y(\tau)$ – температура объекта; $u(\tau)$ – тепловое воздействие; $Y_0$ – температура окружения; $C,RC$ – постоянные коэффициенты.

Дискретные модели системы:
$$\Large y_{\tau+1}=ay_{\tau}+bu_{\tau}$$ (2)
$$\Large y_{\tau+1}=ay_{\tau}-by_{\tau-1}^2+cu_{\tau}+d\sin(u_{\tau-1})$$ (3)

Задача: разработать программу на C++ для моделирования температуры объекта.
