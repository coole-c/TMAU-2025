## Постановка задачи
Необходимо исследовать динамические характеристики температурного объекта, описываемого дифференциальным уравнением:

$$\Large\frac{dy(\tau)}{d\tau}=\frac{u(\tau)}{C}+\frac{Y_0-y(\tau)}{RC} $$ (1)

где $\tau$ – временной параметр; $y(\tau)$ – текущая температура; $u(\tau)$ – управляющее тепловое воздействие; $Y_0$ – температура среды; $C, RC$ – физические константы системы.

В результате дискретизации получены две математические модели:
- **Линейная модель**: $$\Large y_{\tau+1}=ay_{\tau}+bu_{\tau}$$ (2)
- **Нелинейная модель**: $$\Large y_{\tau+1}=ay_{\tau}-by_{\tau-1}^2+cu_{\tau}+d\sin(u_{\tau-1})$$ (3)

где $\tau = 1,2,3{\dots}n$ – дискретные временные интервалы; $a,b,c,d$ – коэффициенты моделей.

**Цель работы**: Разработка программного обеспечения на C++ для анализа температурной динамики объекта по предложенным математическим моделям.
