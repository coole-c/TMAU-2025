## Задание 1. Моделирование управляемого объекта  

Рассмотрим объект, для которого требуется регулировать температуру. Его динамика описывается дифференциальным уравнением:



\[
\frac{dy(\tau)}{d\tau} = \frac{u(\tau)}{C} + \frac{Y_0 - y(\tau)}{RC} \tag{1}
\]



где:  
- \(\tau\) — время,  
- \(y(\tau)\) — температура объекта,  
- \(u(\tau)\) — подводимое тепло,  
- \(Y_0\) — температура окружающей среды,  
- \(C, R\) — постоянные параметры.  

После преобразований получаем дискретные модели:  

**Линейная модель:**



\[
y_{\tau+1} = a y_{\tau} + b u_{\tau} \tag{2}
\]



**Нелинейная модель:**



\[
y_{\tau+1} = a y_{\tau} - b y_{\tau-1}^2 + c u_{\tau} + d \sin(u_{\tau-1}) \tag{3}
\]



где \(\tau = 1,2,3,\dots,n\); \(a, b, c, d\) — константы.  

Задача: реализовать программу на **C++**, моделирующую изменение температуры объекта.
